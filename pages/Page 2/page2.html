Task 2 – CSS Frameworks (Explanation)

1. What is a CSS framework and why is it useful?

A CSS framework is a pre-written collection of CSS styles and components that help developers design websites faster and more efficiently. It provides ready-made layouts, grids, buttons, and responsive design features.

This reduces the need to write CSS from scratch, saving time and improving consistency across projects. CSS frameworks also ensure better compatibility across different browsers and devices.

2. Bootstrap vs Tailwind CSS

Bootstrap is easier to use for beginners because it provides pre-designed components like cards, navbars, and buttons that can be used quickly. However, it is less flexible because designs may look similar across different websites.

Tailwind CSS, on the other hand, is more flexible because it uses utility classes that allow full customization of designs. However, it has a steeper learning curve since developers need to combine many classes to build components.
